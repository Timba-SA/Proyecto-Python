# Operaciones Aritméticas

## Descripción

Escribe un programa en Python que pida al usuario dos números enteros y muestre en pantalla la suma, resta, multiplicación y división de ambos números.

## Entrada

Dos números enteros (en líneas separadas)

## Salida

Cuatro líneas: suma, resta, multiplicación, división

## Ejemplo

**Entrada:**
```
10
2
```

**Salida:**
```
12
8
20
5.0
```

## Restricciones

- Debes usar la función `main()` que lee de stdin y escribe a stdout
- No agregues mensajes de solicitud al usar `input()`
- La salida debe ser exacta según el formato especificado
