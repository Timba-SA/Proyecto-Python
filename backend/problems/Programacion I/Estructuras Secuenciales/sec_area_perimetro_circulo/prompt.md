# Área y Perímetro del Círculo

## Descripción

Escribe un programa en Python que pida al usuario el radio de un círculo y calcule e imprima en pantalla el área y el perímetro del mismo.

## Entrada

El radio del círculo (número decimal)

## Salida

Dos líneas: área y perímetro

## Ejemplo

**Entrada:**
```
5.0
```

**Salida:**
```
78.53981633974483
31.41592653589793
```

## Restricciones

- Debes usar la función `main()` que lee de stdin y escribe a stdout
- No agregues mensajes de solicitud al usar `input()`
- La salida debe ser exacta según el formato especificado
