# Hola Mundo

## Descripción

Crear un programa en Python que imprima en pantalla el mensaje: **"Hola Mundo!"**

## Pautas

- Utiliza la función `print()`
- No necesitas entradas del usuario
- El mensaje debe ser exactamente: `Hola Mundo!` (con mayúsculas y signo de exclamación)

## Ejemplo

**Salida esperada:**
```
Hola Mundo!
```

## Restricciones

- No uses la función `input()`
- La salida debe ser exactamente `Hola Mundo!` sin texto adicional
