# Tabla de Multiplicar

## Descripción

Escribe un programa en Python que solicite al usuario un número entero (n) y luego muestre en pantalla la tabla de multiplicar de ese número desde el 1 hasta el 10, mostrando cada resultado en una línea diferente.

## Entrada

Un número entero

## Salida

10 líneas con formato: {numero} x {multiplicador} = {resultado}

## Ejemplo

**Entrada:**
```
5
```

**Salida:**
```
5 x 1 = 5
5 x 2 = 10
5 x 3 = 15
5 x 4 = 20
5 x 5 = 25
5 x 6 = 30
5 x 7 = 35
5 x 8 = 40
5 x 9 = 45
5 x 10 = 50
```

## Restricciones

- Debes usar la función `main()` que lee de stdin y escribe a stdout
- No agregues mensajes de solicitud al usar `input()`
- La salida debe ser exacta según el formato especificado
