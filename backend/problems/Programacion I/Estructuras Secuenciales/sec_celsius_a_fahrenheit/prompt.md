# Conversión Celsius a Fahrenheit

## Descripción

Escribe un programa en Python que pida al usuario una temperatura en grados Celsius y la convierta a grados Fahrenheit.

## Entrada

Temperatura en grados Celsius (número decimal)

## Salida

{fahrenheit} (resultado de la conversión)

## Ejemplo

**Entrada:**
```
0
```

**Salida:**
```
32.0
```

## Restricciones

- Debes usar la función `main()` que lee de stdin y escribe a stdout
- No agregues mensajes de solicitud al usar `input()`
- La salida debe ser exacta según el formato especificado
