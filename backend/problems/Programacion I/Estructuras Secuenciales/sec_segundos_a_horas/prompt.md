# Conversión de Segundos a Horas

## Descripción

Crea un programa en Python que pida al usuario una cantidad de segundos y muestre cuántas horas equivalen.

## Entrada

Cantidad de segundos (número entero)

## Salida

{segundos} segundos equivalen a {horas} horas

## Ejemplo

**Entrada:**
```
3600
```

**Salida:**
```
3600 segundos equivalen a 1.0 horas
```

## Restricciones

- Debes usar la función `main()` que lee de stdin y escribe a stdout
- No agregues mensajes de solicitud al usar `input()`
- La salida debe ser exacta según el formato especificado
