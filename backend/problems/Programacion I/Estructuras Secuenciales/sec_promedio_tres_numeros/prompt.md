# Promedio de Tres Números

## Descripción

Crea un programa en Python que pida al usuario tres números y calcule e imprima en pantalla el promedio de los mismos.

## Entrada

Tres números (pueden ser enteros o decimales)

## Salida

{promedio} (resultado del promedio)

## Ejemplo

**Entrada:**
```
10
20
30
```

**Salida:**
```
20.0
```

## Restricciones

- Debes usar la función `main()` que lee de stdin y escribe a stdout
- No agregues mensajes de solicitud al usar `input()`
- La salida debe ser exacta según el formato especificado
